## Overview

DevFlows is a full-stack project I worked on as a freelancer, focused on an
AI-powered cloud IDE with real-time workflows and multi-file project support.
The experience is intentionally minimal, with a dark workspace-first layout
and clear entry points for starting a new project or importing an existing one.

## What stands out

- Architected an event-driven backend with async orchestration, cancellation
  handling, and fault-tolerant execution.
- Reduced latency using debounced inference and optimized autosave under
  continuous user activity.
- Accelerated GitHub repository ingestion through parallel codebase parsing and
  batch file processing.
- Designed a scalable multi-tenant architecture with secure access control and
  efficient indexed queries.

## Page structure

- Centered product branding with a focused launch state.
- Two quick actions for creating a new workspace or importing one.
- Minimal spacing and dark surfaces to reinforce the IDE aesthetic.
- Empty surrounding canvas that keeps attention on the core IDE entry points.
