## Overview

Phone Wraps is a scalable custom phone cover e-commerce platform I built as a
full-stack engineer. The experience lets users design, personalize, and
purchase device-specific products through a responsive Next.js interface while
keeping the customization flow fast and reliable.

## My role

As a full-stack engineer, I worked across the UI, API layer, and order
workflow to deliver a production-ready shopping experience.

## What I focused on

- Built dynamic customization workflows with image uploads, template rendering,
  and high-resolution print previews.
- Developed product discovery for 350+ device models with optimized filtering,
  search, and personalization that improved navigation across large SKU sets and
  increased conversion rates by 25%.
- Implemented secure checkout and payment gateway integration with Stripe,
  along with end-to-end order management, inventory tracking, and automated
  email notifications.
- Designed backend workflows for order processing and fulfillment across
  pan-India shipping operations.
- Optimized REST APIs and PostgreSQL queries, reducing average page load time by
  60% while minimizing rendering inconsistencies.

## Notes

The project focused on turning a high-SKU commerce experience into a smoother,
more reliable flow for customers and operations teams alike.
